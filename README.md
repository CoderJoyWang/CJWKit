# CJWKit

v1.0
CJWKit     包含一些数据请求、处理、存储的工具，基于sqlite3.0的数据库读写工具、基于NSURLSession的网络请求工具、基于runtime的json转model工具
CJWView    包含一些用的比较多的控件，UICollectionView和UITableView的无数据页面、视图加载中的加载页面、'不可重用'的tableview(适用于写个人中心、设置等cell的样式基本不同的页面)
